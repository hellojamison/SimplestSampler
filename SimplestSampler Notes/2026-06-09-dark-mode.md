# Dark mode preference

## Goal

Add System / Light / Dark appearance toggle in Settings, matching OverCue sampler theme behavior.

## Changes

- `SamplerThemeMode` + `SamplerThemeColors` with OverCue light/dark sampler CSS values
- `SamplerThemedRoot` environment injection for main + settings windows
- `themeMode` on `SamplerPreferences` with legacy `darkModeEnabled` migration
- Appearance section in `ShortcutPreferencesView`
- Views/button styles read `@Environment(\.samplerThemeColors)`
