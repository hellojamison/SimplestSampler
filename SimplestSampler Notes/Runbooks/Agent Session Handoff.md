# Agent Session Handoff

End-of-session checklist for agents and contributors:

1. Update `SimplestSampler Notes/YYYY-MM-DD.md` with `- HH:MM:` bullets (Working / Not verified / Tried and failed).
2. Create or update a task note if work was substantive.
3. Update [[Home]] **Current state** (phase, blockers, next).
4. Touch [[Phases/Phase 1 Playback]] or [[Phases/Phase 2 PT Capture]] checklists when criteria change.
5. Update [[Areas/*]] when long-lived behavior changes.
6. Do not commit secrets. Do not edit OverCue files.
